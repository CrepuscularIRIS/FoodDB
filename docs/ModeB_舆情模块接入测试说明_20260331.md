# ModeB 舆情模块接入测试说明（2026-03-31）

## 已接入内容

1. 新增 MediaCrawler 舆情导入模块：`backend/opinion_module.py`
2. 新增导入脚本：`scripts/import_mediacrawler_opinion.py`
3. 新增后端接口：
- `POST /modeb/opinion/import`
- `GET /modeb/opinion/summary`
- `GET /modeb/opinion/top`
4. ModeB 症状评估接口已融合舆情特征：
- `POST /symptom/assess`
- `POST /symptom/assess_stream`

## 一键导入（示例）

```bash
curl -X POST 'http://127.0.0.1:8000/modeb/opinion/import' \
  -H 'Content-Type: application/json' \
  -d '{
    "media_root": "/home/yarizakurahime/Agents/winteragent/MediaCrawler/data",
    "platform": "all",
    "days": 365
  }'
```

## 查看导入摘要

```bash
curl 'http://127.0.0.1:8000/modeb/opinion/summary'
```

## 查看舆情风险 Top 企业

```bash
curl 'http://127.0.0.1:8000/modeb/opinion/top?top_n=10'
```

## 验证 ModeB 已吃到舆情输入

```bash
curl -X POST 'http://127.0.0.1:8000/symptom/assess' \
  -H 'Content-Type: application/json' \
  -d '{"query":"腹泻 发热 呕吐"}'
```

检查返回里 `linked_enterprises` 的新增字段：
- `opinion_risk_index`
- `opinion_mentions_30d`
- `opinion_negative_ratio_30d`
- `opinion_risk_keyword_hits_30d`
- `combined_risk_score`
