## 2. 数据模型设计

### 2.1 节点类型定义 (Node Types)

| 节点类型 | 数量(建议) | 描述 | 关键属性 |
|---------|-----------|------|---------|
| `牧场` | 100-200 | 生鲜乳来源 | 养殖规模、卫生等级、挤奶设备 |
| `乳企` | 50-100 | 乳制品加工 | 产能、认证情况、违规记录 |
| `物流` | 80-150 | 冷链运输 | 冷链覆盖率、运输距离、时效性 |
| `仓储` | 60-120 | 仓储存储 | 温控能力、库存周转、清洁度 |
| `零售` | 100-200 | 终端销售 | 销售规模、存储条件、投诉率 |
| `批次` | 110-230 | 产品批次 | 生产日期、保质期、检验结果 |

**节点总数**: 500-1000 节点

### 2.2 边关系定义 (Edge Types)

| 边类型 | 源节点 | 目标节点 | 描述 | 权重因子 |
|--------|--------|---------|------|---------|
| `supply` | 牧场 | 乳企 | 原料供应关系 | 供应量占比 |
| `process` | 乳企 | 乳企 | 加工委托关系 | 加工深度 |
| `transport` | 乳企 | 物流 | 运输委托关系 | 运输频次 |
| `storage` | 物流 | 仓储 | 仓储入库关系 | 存储时长 |
| `sale` | 仓储 | 零售 | 分销关系 | 分销范围 |
| `trace` | 批次 | 任意 | 追溯关联 | 关联强度 |

## 3. 模型架构

### 3.1 整体模型设计

```
┌─────────────────────────────────────────────────────────────────────────┐
│                        异构图风险预测模型 (GNN)                          │
└─────────────────────────────────────────────────────────────────────────┘

输入层
  │
  ▼
┌─────────────────────────────────────────────────────────────────────────┐
│  节点特征嵌入 (Node Feature Embedding)                                  │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │ 类别特征 → Embedding (32维)                                     │   │
│  │ 数值特征 → Linear + LayerNorm (32维)                             │   │
│  │ 聚合 → Concat → Linear(64→128)                                 │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                              │                                          │
│                              ▼                                          │
┌─────────────────────────────────────────────────────────────────────────┐
│  图神经网络层 (Graph Neural Network)                                    │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │ Graph Attention Network (3层)                                  │   │
│  │   - Multi-head attention (4 heads)                             │   │
│  │   - Dropout (0.2)                                              │   │
│  │   - LayerNorm                                                  │   │
│  │   - Residual connection                                        │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                              │                                          │
│                              ▼                                          │
┌─────────────────────────────────────────────────────────────────────────┐
│  异构图聚合 (Heterogeneous Aggregation)                                  │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │ 类型感知注意力 (Type-aware Attention)                           │   │
│  │   - 不同节点类型使用不同的注意力权重                            │   │
│  │   - 边类型加权聚合                                              │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                              │                                          │
│                              ▼                                          │
┌─────────────────────────────────────────────────────────────────────────┐
│  输出头 (Output Heads)                                                  │
│  ┌──────────────────┐  ┌──────────────────┐  ┌──────────────────┐       │

│  │  Sigmoid(0-1)    │  │  低/中/高        │  │  不确定性估计    │       │

└─────────────────────────────────────────────────────────────────────────┘
```

### 3.2 回归模型详细设计

#### 3.2.1 模型参数

| 组件 | 参数 | 值 |
|------|------|-----|
| 节点特征维度 | input_dim | 64 |
| 嵌入维度 | hidden_dim | 128 |
| GNN层数 | num_layers | 3 |
| 注意力头数 | num_heads | 4 |
| Dropout | dropout | 0.2 |
| 学习率 | lr | 0.001 |
| 批次大小 | batch_size | 32 |

#### 3.2.2 损失函数

```python
# 多任务学习损失
total_loss = α * risk_prob_loss + β * risk_level_loss + γ * confidence_loss

# 风险概率回归 (回归任务)
risk_prob_loss = MSE(pred_prob, true_prob)
# 目标: 预测 0-1 之间的连续概率值

# 风险等级分类 (辅助任务)
risk_level_loss = CrossEntropy(pred_level, true_level)
# 类别: 低(0), 中(1), 高(2)

# 置信度估计 (不确定性任务)
confidence_loss = MSE(pred_confidence, uncertainty_measure)
# 使用 MC Dropout 估计不确定性
```

### 3.3 训练流程

```
┌─────────────────────────────────────────────────────────────────────────┐
│                           训练流程 (Training Pipeline)                   │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────┐
│  数据准备        │
│  (500-1000节点) │
└────────┬────────┘
         │
         ▼
┌─────────────────────────────────────────────────────────────────────────┐
│  Step 1: 数据预处理                                                     │
│  - 缺失值填充 (均值/中位数/插值)                                        │
│  - 类别特征编码 (One-hot / Label Encoding)                              │
│  - 数值特征标准化 (StandardScaler)                                     │
│  - 图结构构建 (NetworkX)                                               │
└────────┬────────────────────────────────────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────────────────────────────────────────┐
│  Step 2: 特征工程                                                      │
│  - 节点特征聚合                                                        │
│  - 边特征构建                                                          │
│  - 图结构特征 (PageRank, 中心性, 子图模式)                              │
│  - 时序特征 (滑动窗口)                                                 │
└────────┬────────────────────────────────────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────────────────────────────────────────┐
│  Step 3: 图划分                                                        │
│  - 训练集 / 验证集 / 测试集 (8:1:1)                                    │
│  - 节点划分 (保持图连通性)                                             │
│  - 边划分 (避免信息泄露)                                               │

         │
         ▼
┌─────────────────────────────────────────────────────────────────────────┐
│  Step 4: 模型训练                                                      │
│  for epoch in range(max_epochs):                                       │
│    for batch in dataloader:                                            │
│      # 前向传播                                                        │
│      risk_prob, risk_level, confidence = model(graph, node_features)   │

│      if epoch % 10 == 0:                                               │

         │
         ▼

│  Step 5: 模型评估                                                      │
```

### 3.4 评估指标体系

| 指标 | 目标值 | 计算方式 | 业务含义 |
|------|--------|---------|---------|
| **Recall@95%** | ≥95% | 在预测为高风险的样本中，覆盖95%真实高风险 | 尽可能不漏掉高风险企业 |
| **F1 Score** | ≥93% | 2×Precision×Recall/(Precision+Recall) | 综合预测准确性 |
| **Precision** | ≥92% | TP/(TP+FP) | 预测高风险的准确性 |
| **AUC-ROC** | ≥0.95 | ROC曲线下面积 | 区分能力 |
| **Confidence Calibration** | ≤0.05 | ECE (Expected Calibration Error) | 置信度与准确率匹配度 |

**风险阈值设定**:
```
概率 < 0.3  → 低风险 (常规监管)
0.3 ≤ 概率 < 0.7  → 中风险 (加强抽检)
概率 ≥ 0.7  → 高风险 (专项检查+现场核查)
```

## 8. 风险与挑战

| 风险 | 影响 | 缓解措施 |
|------|------|---------|
| 异构图表示学习复杂度高 | 模型训练慢 | 使用简化的GAT架构 |
| 小样本风险标注困难 | 模型泛化差 | 结合规则生成伪标签 |
| 实时流式性能要求 | 前端卡顿 | 使用WebWorker/分片加载 |
| 500节点图可视化密集 | 渲染性能差 | 使用LOD (Level of Detail) |
| 目标指标要求高 | 难以达标 | 集成多模型/集成学习 |
